üìñ Define HTML

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines the layout of web pages using a variety of elements or tags it was developed by Tim Berners-Lee in 1991.


Tags are classified into:

  • Empty Tag
    •

    Empty tags,also called self-closing tags,do not have closing tags.

    Examples:<br>,<hr>,<img>,<link>

  • Non-Empty Tag
    •

    Non-Empty Tags,also called container tags or paired tags,have both opening and closing tags.

    Examples:<p>,<div>,<h1>,<ul>


HTML lists and their types with examples.

HTML lists can be:

  • Ordered list: <ol> for numbered lists.
  • Unordered list: <ul> for bullet points.
  • Description list: <dl> with <dt> and <dd>.

üìñ Define CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and design of a web page. It controls how HTML elements are displayed on screen, paper, or other media by specifying styles such as colors, fonts, spacing, and layout it was developed by H√•kon Wium Lie in 1996.


Types of CSS

There are three main types of CSS:

  • Inline CSS: Applied directly to an HTML element using the style attribute.
  • Internal CSS: Written inside a <style> tag within the HTML document's head.
  • External CSS: Written in a separate .css file and linked to the HTML using <link>.

<p style="color:red;">Inline CSS Example</p>

<style>
p { color: blue; }
</style>

<link rel="stylesheet" href="style.css">

Advantages & Disadvantages of CSS

Advantages: Separation of design and content, reusability, faster page loading.

Disadvantages: Browser compatibility issues, complexity in large projects.


CSS Selectors

Selectors define which HTML elements CSS rules apply to:

  • Element Selector: p { color: blue; }
  • Class Selector: .className { color: green; }
  • ID Selector: #idName { color: red; }
  • Universal Selector: * { margin: 0; }
  • Attribute Selector: input[type=text] { border: 1px solid; }

CSS Positioning

CSS positioning types:

  • Static - Default positioning.
  • Relative - Positioned relative to its normal position.
  • Absolute - Positioned relative to the nearest positioned ancestor.
  • Fixed - Stays in the same position even when scrolling.
  • Sticky - Switches between relative and fixed depending on scroll.

Pseudo-classes & Pseudo-elements

Pseudo-class: a:hover { color: red; }

Pseudo-element: p::first-letter { font-size: 24px; }

9. CSS Units

Absolute (px, cm, in) and Relative (%, em, rem, vh, vw) units are used in CSS.

10. Transitions & Animations


.box {
    transition: background 0.5s;
}
.box:hover {
    background: red;
}

Background Properties

Examples:


background-color: blue;
background-image: url('image.jpg');
background-size: cover;

Inline-block, Block, Inline

  • Block: Takes full width (div).
  • Inline: Takes only content width (span).
  • Inline-block: Behaves like inline but allows block styling.

Class and Object

A class is like a blueprint or design.
An object is a real thing made from that class.

Example

 
//Creating a class
class Car {
    void drive() {
        System.out.println("Driving...");
    }

    public static void main(String[] args) {
        Car myCar = new Car();   // Creating an object of Car
        myCar.drive();           // Calling the drive method using the object
    }
}

Data Types

Java has different types of data:

  • int - for whole numbers (e.g., 10)
  • float - for decimal numbers (e.g., 10.5)
  • char - for single characters (e.g., 'A')
  • boolean - for true or false
  • String - for words or text (e.g., "Hello")

Input and Output

  • Input means taking data from the user (e.g., name, age, number) through the keyboard. Java uses the Scanner class (from java.util package) to read input.
  • Output means displaying data to the user using the console. Java uses System.out.print() or System.out.println() to show output.

Example


import java.util.Scanner;

class InputOutputExample {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);  // Create Scanner object

        System.out.print("Enter your name: ");
        String name = input.nextLine();          // Taking string input

        System.out.print("Enter your age: ");
        int age = input.nextInt();               // Taking integer input

        // Output printing
        System.out.println("Hello, " + name + "!"); 
        System.out.println("You are " + age + " years old.");
    }
}

üìñ Define SQL

SQL (Structured Query Language) is a standard programming language used to manage and manipulate relational databases. It is used for tasks such as inserting, updating, deleting, and retrieving data, as well as defining and controlling database structures. SQL was developed in the early 1970s by Donald D. Chamberlin and Raymond F. Boyce at IBM.


Types of statements

  • DDL (Data Definition Language) ‚Äì CREATE, ALTER, DROP
  • DML (Data Manipulation Language) ‚Äì INSERT, UPDATE, DELETE
  • DQL (Data Query Language) ‚Äì SELECT
  • DCL (Data Control Language) ‚Äì GRANT, REVOKE
  • TCL (Transaction Control Language) ‚Äì COMMIT, ROLLBACK, SAVEPOINT

difference between PRIMARY KEY and UNIQUE KEY

  • PRIMARY KEY ‚Äì Uniquely identifies each record in a table, cannot have NULL values.
  • UNIQUE KEY ‚Äì Ensures all values in a column are unique, but can have one NULL value.
 
        CREATE TABLE Student (
            ID INT PRIMARY KEY,
            Email VARCHAR(50) UNIQUE
        );

types of JOINS in SQL with examples

  • INNER JOIN ‚Äì Returns records with matching values in both tables.
  • LEFT JOIN ‚Äì Returns all records from the left table and matching from the right.
  • RIGHT JOIN ‚Äì Returns all records from the right table and matching from the left.
  • FULL JOIN ‚Äì Returns all records when there is a match in one of the tables.
 
        SELECT s.name, c.course
        FROM Students s
        INNER JOIN Courses c ON s.id = c.student_id;

Normalization

Normalization is the process of organizing data to reduce redundancy. Types:

  • 1NF ‚Äì Remove repeating groups.
  • 2NF ‚Äì Remove partial dependencies.
  • 3NF ‚Äì Remove transitive dependencies.
  • BCNF ‚Äì Every determinant is a candidate key.

Difference between DELETE, TRUNCATE, and DROP.

  • DELETE ‚Äì Removes rows based on condition, can be rolled back.
  • TRUNCATE ‚Äì Removes all rows, cannot be rolled back.
  • DROP ‚Äì Deletes entire table structure and data.

SQL constraints

  • NOT NULL
  • UNIQUE
  • PRIMARY KEY
  • FOREIGN KEY
  • CHECK
  • DEFAULT

Stored Procedures with examples

Stored procedures are precompiled SQL code stored in the database.

 
        CREATE PROCEDURE GetEmployees
        AS
        SELECT * FROM Employees;

üìñ Define C++

C++ is an object-oriented programming language developed by Bjarne Stroustrup. It is an extension of the C language with features like classes, objects, inheritance, and polymorphism it was developed by Bjarne Stroustrup in 1985.


Features of C++?

  • Object-Oriented Programming (OOP)
  • Simple and Easy to Learn
  • Compiled and Fast
  • Portability
  • Memory Management
  • Rich Standard Library (STL)
  • Extensibility
  • Multi-Paradigm Language

class and Object in C++

  • A class is a user-defined data type that contains data members and member functions to perform operations on the data.
  • An object is an instance of a class. It is created to access the class members (data and functions).

Constructor in C++

A constructor is a special function with the same name as the class that is automatically called when an object is created.


What is a destructor in C++?

A destructor is a special function that is automatically called when an object is destroyed to release resources.


What is inheritance in C++?

Inheritance allows a class to inherit properties and behaviors from another class, promoting code reuse.


What is polymorphism in C++?

Polymorphism allows one function to behave differently based on the object or input.


Encapsulation

Encapsulation is the concept of hiding internal data and showing only necessary information. It is implemented using access specifiers.


Access specifiers in C++

  • Public: Accessible from anywhere
  • Private: Accessible only within the class
  • Protected: Accessible within the class and its derived classes.

What is function overloading?

Function overloading means having multiple functions with the same name but different parameters within the same scope.


What is operator overloading?

Operator overloading allows us to redefine the meaning of an operator (like +, -) for user-defined types.


What is a virtual function?

A virtual function is a member function that is declared using the keyword 'virtual' and is used for achieving run-time polymorphism.

What is recursion?

A function calling itself.
Example

#include 

// Recursive function to calculate factorial
int factorial(int n) {
    if (n == 0)
        return 1;  // Base case
    else
        return n * factorial(n - 1);  // Recursive call
}

int main() {
    int number;
    printf("Enter a number: ");
    scanf("%d", &number);

    int result = factorial(number);
    printf("Factorial of %d is %d", number, result);

    return 0;
}
